# Flight Management System ✈️

A Django-based Flight Management System that allows users to:

- View available flights
- Search flights
- Login authentication
- Book flights
- View booking history
- Experience modern UI with video background

## Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite

## Features

- User Login System
- Flight Booking
- Booking History
- Protected Routes
- Dynamic Flight Cards
- Flight Timings
- Background Video UI

## Future Improvements

- Payment Gateway
- Seat Availability
- Ticket PDF Download
- Flight Search Optimization

```bash
python manage.py runserver
